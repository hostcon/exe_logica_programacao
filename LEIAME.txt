# 📚 Estrutura sugerida do curso (40h)

## 🔹 Aula 1 — Introdução à lógica + ambiente

* O que é lógica de programação

* Instalação (VS Code + Python)
* Primeiro programa (`print`)
* Entrada de dados (`input`)

💡 Exercícios:

* Mostrar nome, idade
* Soma de dois números

## 🔹 Aula 2 — Variáveis e operadores (seu `a1_variaveis_operadores`)

* Operadores relacionais
* Conversão de tipos (`int()`, `float()`)

💡 Exercícios:

* Média de aluno
* Conversão de metros → cm

## 🔹 Aula 3 — Estruturas condicionais (seu `a2_estruturas_condicionais`)

# 🎯 Dica didática (ouro para leigos)

* Ensine sempre assim:

  1. Problema do mundo real
  2. Lógica (no papel)
  3. Código em Python

* Evite:
  ❌ Explicar sintaxe antes da lógica
  ❌ Termos técnicos demais no início

* Use muito:
  ✔ Exemplos do dia a dia
  ✔ Exercícios curtos e progressivos
  ✔ Repetição (isso fixa)
